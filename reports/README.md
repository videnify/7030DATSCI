# reports/

**Purpose:** All generated, non-code outputs of the pipeline — figures, tables, per-model statistics summaries, and the dissertation itself.
**Added:** 2026-07-06 (repository hygiene pass — this folder previously had no top-level README; its subfolders are self-explanatory by name but were not indexed anywhere in one place).

## Contents

| Subfolder | Purpose |
|-----------|---------|
| `dissertation/` | The dissertation document itself (tracked; the only `reports/` content meant to be a final, citable deliverable) |
| `figures/` | All chart/plot PNGs (300 dpi where publication-quality) regenerated by the numbered notebooks — gitignored, reproducible |
| `tables/` | CSV exports of statistical/descriptive tables (correlation matrices, stationarity tests, metrics) referenced by the dissertation |
| `statistics/` | Per-mission Markdown summaries (e.g. `06_baseline_summary.md`, `07_event_models_summary.md`, `07A`/`07B`/`07C` isolated-experiment summaries) — narrative companions to the tables/figures, not raw data |
| `baseline/` | `Baseline_LASSO` (MCP v1.0) outputs: `baseline_metrics.json`, `baseline_predictions.parquet` |
| `event/` | Isolated single-model re-run outputs (Mission 07A/07B/07C experiments) — descriptive cross-checks, not the primary RQ3 comparison |
| `model_comparison/` | The primary, frozen RQ3 comparison (MCP v1.0): `model_comparison.parquet`, `statistical_tests.json`, `feature_importance.parquet`, `shap_values_*.parquet`, `event_model_predictions.parquet` |
