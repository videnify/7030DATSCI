# data/raw/

**Updated 2026-07-06** — corrected to match what `01_data_collection.ipynb` actually produces; the previous version of this file described files (`fred_macro.parquet`, `gdelt_events.parquet`) that were never the actual output filenames.

Untouched source data — never modify files in this directory.

| File | Source | Description |
|------|--------|-------------|
| `prices.parquet` | yfinance | Daily adjusted close for SPY, QQQ, GLD, TLT (2015–2025) — QQQ/GLD/TLT are optional, exploratory-only datasets |
| `spy_ohlcv.parquet` | yfinance | Full OHLCV for SPY (used in the event study) |
| `spy_returns.parquet` | yfinance (derived) | SPY log/simple returns and next-day prediction targets, derived from `spy_ohlcv.parquet` in the same collection step |
| `vix.parquet` | yfinance | VIX daily closes, changes, and a high-VIX regime flag (2015–2025) |
| `macro_indicators.parquet` | FRED API | Federal Funds Rate, CPI, Unemployment Rate, 10Y/2Y Treasury yields, yield spread, manufacturing employment |
| `app_presidential_documents_raw.parquet` | American Presidency Project | Raw presidential documents (speeches, executive orders, press conferences, proclamations) across the four core administrations |
| `app_presidential_documents_economic.parquet` | American Presidency Project (filtered) | Subset of the above narrowed to economically/politically relevant communications by keyword filter |
| `app_presidential_documents.parquet` | American Presidency Project (legacy alias) | Retained on disk from an earlier collection pass; `app_presidential_documents_economic.parquet` is the file actually consumed downstream |
| `fomc_dates.parquet` | Federal Reserve (manual) | FOMC meeting dates with rate decisions (2015–2025) |
| `gdelt_sample.parquet` | GDELT 2.0 | 5-day proof-of-concept sample of geopolitical event data — **not** a full-history series, see `docs/research_bible/11_limitations.md` L7 |
| `daily_modelling_calendar_v1.parquet` | Derived | First aligned daily calendar joining returns/VIX/macro/FOMC/presidential-event flags — an early collection-stage checkpoint, superseded downstream by `data/processed/master_dataset.parquet` |
| `data_quality_report.parquet` | Derived | Row counts, date ranges, and missing-value summary per collected source, for the Phase 1 data-quality check |

Generated by: `notebooks/01_data_collection.ipynb`.
